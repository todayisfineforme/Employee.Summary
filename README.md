# Employee Summary
CLI utilizing inquirer that asks for employee details and builds an employee roster in an output html file.

## Installation

clone repo then 

```bash
npm install
```

## Usage

```node.js
npm start
```

answer prompts for all company employees. once complete refer to 'team.html' in output dir.
